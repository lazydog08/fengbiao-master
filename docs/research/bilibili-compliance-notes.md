# B 站公开数据采集边界

这份项目默认只研究公开视频的公开页面和公开元数据。实现时优先选择稳定、低频、可追溯的方式获取信息，并在失败时记录原因。

## 可以纳入的数据

- 视频标题
- 视频封面公开 URL
- 视频链接、BV 号、AV 号
- UP 主公开名称与 mid
- 发布时间、分区、标签
- 播放、点赞、投币、收藏、分享、评论、弹幕等公开视频指标

## 不纳入的数据

- cookie、登录态、浏览器 profile
- 私信、未公开内容、会员专属内容
- 个人隐私数据
- 绕过验证码或风控得到的数据
- 与研究封标无关的大规模评论原文

## 工程要求

- 所有请求设置清晰 User-Agent。
- 对同一域名做限速和指数退避。
- 每次采集记录 `source_url`、`fetched_at`、`fetch_method`。
- 抓取失败要可重试，但不能无限重试。
- 采集器和分析器分离，避免分析失败影响数据入库。
