# 封标大师

封标大师是一个面向内容创作者的 B 站封面与标题研究工具。它会围绕一份博主清单，定期记录公开视频的封面、标题、发布时间和播放量变化，逐步建立“封标知识库”，帮助判断什么样的选题、标题表达和封面元素更容易获得点击。

## 目标

1. 维护一个“值得学习的博主清单”。
2. 定期抓取这些博主的新视频公开元数据。
3. 保存封面、标题、播放量和互动指标的时间序列快照。
4. 对封面和标题做结构化标注与归类。
5. 未来输入一个选题时，能基于历史样本给出封面与标题建议。

## MVP 边界

- 输入：博主清单、采集频率、每位博主的新视频范围。
- 输出：本地知识库、可检索样本、基础分析报告。
- 数据：只使用公开视频公开信息，例如标题、封面地址、播放量、发布时间、UP 主、视频链接。
- 不做：登录态采集、弹幕/评论深挖、自动发布、刷量判断、绕过平台限制。
